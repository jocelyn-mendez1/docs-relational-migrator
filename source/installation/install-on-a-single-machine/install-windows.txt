===============
Install Windows
===============

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

You can install Relational Migrator on a single Windows machine. This installation
method uses default settings and binds Relational Migrator to a single 
IP and port. 

Steps
-----

.. procedure::
   :style: normal

   .. step:: Download the latest Windows Relational Migrator binary from the 
   `release page <https://migrator-installer-repository.s3.ap-southeast-2.amazonaws.com/index.html>`__.

   .. step:: Double-click the ``MongoDB Relational Migrator-X.X.X.msi`` file.

   .. step:: When you are prompted with the MongoDB Relational Migrator Setup 
      Wizard, click :guilabel:`Next`.

     Accept the license agreement and click :guilabel:`Next` to continue.

   .. step:: Click :guilabel:`Install` to start your Migrator installation.

   .. step:: Click :guilabel:`Finish` to exit the installer.

   .. step:: (Optional) To connect to Oracle or MySQL drivers, download and place 
    the drivers into your Relational Migrator directory.

   .. step:: To download and install MySQL or Oracle drivers, see the following:

    - `MySQL <https://dev.mysql.com/downloads/connector/j/>`__

    - `Oracle <https://www.oracle.com/database/technologies/appdev/jdbc-downloads.html>`__

   .. step:: Copy the driver files to ``/Library/Application Support/MongoDB/  Relational Migrator/Drivers``.

   .. note::

   Relational Migrator installs the JDBC drivers for SQL Server and PostgreSQL by 
   default.
