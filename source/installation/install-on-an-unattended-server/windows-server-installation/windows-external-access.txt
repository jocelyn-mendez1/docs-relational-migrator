========================================================
Enable External Access to Relational Migrator on Windows
========================================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

Enable external access to Relational Migrator on Windows. By enabling external 
access, incoming connections are allowed on the same port as Relational Migrator. 

Steps
-----

.. procedure::
   :style:normal

   .. step:: Open Windows Defender Firewall with Advanced Security.

   .. step:: Select :guilabel:`Inbound Rules`. Click :guilabel:`New Rule`.

   .. step:: Select :guilabel:`Port`. Click :guilabel:`Next`.

   .. step:: Select :guilabel:`TCP`, :guilabel:`Specific Local Ports`, and enter the port 
    number. Click :guilabel:`Next`. 

   .. step:: Select :guilabel:`Allow the connection`. Click :guilabel:`Next`.

   .. step:: Select the desired network profiles. Click :guilabel:`Next`. 

   .. step:: Enter a name for the rule. Click :guilabel:`Finish`.
